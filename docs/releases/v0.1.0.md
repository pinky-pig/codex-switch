# v0.1.0

首个可用版本。

## 包含内容

- Node.js CLI / TUI
- macOS 原生菜单栏应用
- 多账号保存、切换、删除
- 从菜单栏直接打开 CLI
- 打开 `auth.json` / `config.toml`

## Release 产物

- `Codex-Switch-macOS.zip`

解压后包含：

- `Codex Switch.app`

## 使用方式

1. 下载 release 产物
2. 解压 `Codex-Switch-macOS.zip`
3. 双击 `Codex Switch.app`
4. 顶部菜单栏出现图标后即可使用

## 说明

- 切换账号后，需要重启当前 VS Code / Codex / 终端会话，新的账号才会生效
- 账号快照默认保存在 `~/.codex-switch/accounts/`
